# Portfolio — Étudiant Supinfo B3 Informatique

Un portfolio moderne et responsive pour un étudiant en 3ème année de Bachelor Informatique à Supinfo.

## Aperçu

Portfolio single-page avec un design dark theme élégant, des animations fluides et une expérience utilisateur soignée.

## Technologies utilisées

- **HTML5** — Structure sémantique
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript** — Vanilla JS, Intersection Observer API
- **Font Awesome** — Icônes
- **Google Fonts** — Inter & Fira Code

## Fonctionnalités

- Design responsive (mobile-first)
- Dark theme moderne avec accent indigo/violet
- Effet typewriter dans le hero
- Animations au scroll (Intersection Observer)
- Compteur animé pour les statistiques
- Effet de glow suivant le curseur
- Navigation avec détection de section active
- Formulaire de contact avec animation de soumission
- Menu hamburger pour mobile
- Smooth scroll

## Sections

1. **Hero** — Introduction avec effet typewriter
2. **À propos** — Présentation, stats et détails
3. **Compétences** — Frontend, Backend, BDD, DevOps, Outils, Soft skills
4. **Projets** — 6 projets avec descriptions et technologies
5. **Formation** — Timeline avec parcours académique et professionnel
6. **Contact** — Formulaire et méthodes de contact

## Installation

```bash
# Cloner le repository
git clone https://github.com/votre-username/portfolio.git

# Ouvrir index.html dans un navigateur
# Ou utiliser un serveur local :
npx serve .
```

## Personnalisation

1. Modifier les informations personnelles dans `index.html`
2. Ajuster les couleurs dans les CSS variables (`css/style.css`)
3. Remplacer le placeholder de photo par votre image
4. Mettre à jour les liens sociaux et projets

## Structure du projet

```
portfolio/
```
